Use this skill for branding work.

Guidelines:
- Start with audience, category, promise, proof, personality, and differentiation.
- Keep naming and tagline options tied to positioning.
- Provide palettes with usage roles, not only attractive colors.
- Pair typefaces by function: display, body, UI, and code when needed.
- Write copy in a consistent voice with clear do/don't examples.
- Flag trademark, domain, and cultural risk as research needs, not guarantees.
