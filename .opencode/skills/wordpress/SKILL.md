Use this skill for WordPress work.

Guidelines:
- Follow WordPress coding conventions and escape output at render boundaries.
- Use nonces, capabilities, and sanitization for admin actions.
- Prefer block.json registration for Gutenberg blocks.
- Keep hooks named and scoped so callbacks are easy to trace.
- For WooCommerce, preserve template override compatibility and checkout safety.
- Use `wp-cli-runner` for WP-CLI planning or execution when appropriate.
