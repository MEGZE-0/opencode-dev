## Konfiguration

Du kan aktivere og tilpasse LSP-servere gennem sektionen `lsp` i din nexusflow-konfiguration.

For at aktivere alle indbyggede LSP-servere skal du indstille `lsp` til `true`.

```json title="nexusflow.json"

  "$schema": "https://nexusflow.ai/config.json",

```


### Deaktivering af LSP-servere

Hvis `lsp` udelades, er alle LSP-servere deaktiveret. For at deaktivere alle LSP-servere efter at en anden konfiguration har aktiveret dem, skal du indstille `lsp` til `false`:

```json title="nexusflow.json" {3}
{
  "$schema": "https://nexusflow.ai/config.json",
  "lsp": false
}
```

For at deaktivere en **specifik** LSP-server skal du indstille `disabled` til `true`:

```json title="nexusflow.json" {5}
{
  "$schema": "https://nexusflow.ai/config.json",
  "lsp": {
    "typescript": {
      "disabled": true
    }
  }
}
```
