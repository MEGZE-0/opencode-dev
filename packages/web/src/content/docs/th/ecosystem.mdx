ชุดโครงการชุมชนที่สร้างขึ้นบน NexusFlow

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ NexusFlow ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-nexusflow](https://github.com/awesome-nexusflow/awesome-nexusflow) และ [nexusflow.cafe](https://nexusflow.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
